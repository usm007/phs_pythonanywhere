Headers: roll_no,name or class_name,roll_no,name
How should the Excel/CSV file look?

Step 1: Create the sheet in Excel with the exact header names in row 1.

Step 2: Fill student rows below the header (no blank required columns).

Step 3: Save as CSV UTF-8 (Comma delimited) (*.csv), then import here.

Class format accepted in CSV: You can use either full class names or short codes.

Accepted class_name values:
Class 6, Class 7, Class 8(A), Class 8(B), Class 9(A), Class 9(B)

OR short style:
6, 7, 8a, 8b, 9a, 9b
also accepted: class6, class7, class8a, class8b, class9a, class9b

Single-class format (select class above):

roll_no,name
1,Aman Das
2,Riya Sharma
3,Neel Bora

Multi-class format (leave class blank above):

class_name,roll_no,name
class_6,1,Aman Das
class_7,12,Riya Sharma
class_8,4,Neel Bora
  • Use commas as separators.
  • Accepted header aliases: roll_no/roll, name/student_name, class_name/class.
  • For multi-class imports, class_name also accepts 6/7/8a/8b/9a/9b style values.
  • If a row has missing values, it will be skipped.
  • For existing roll numbers in a class, name will be updated.
  • If the same class+roll appears multiple times in one CSV, the last row is used.
